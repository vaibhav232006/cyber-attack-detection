# 🔐 Cyber Attack Detection System

## 📌 Overview

This project focuses on detecting cyber attack patterns using **Unsupervised Machine Learning (Isolation Forest)**. It analyzes network traffic data and identifies unusual patterns that may indicate cyber attacks.

---

## 🎯 Objectives

* Detect anomalies in network traffic
* Identify potential cyber attacks
* Visualize results using a web interface

---

## 🧠 Methodology

1. Data Collection (NSL-KDD dataset)
2. Data Preprocessing (encoding + scaling)
3. Apply Isolation Forest algorithm
4. Detect anomalies
5. Display results using Streamlit

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

---

## 📊 Output

* `1` → Normal traffic
* `-1` → Anomaly (Possible attack)
